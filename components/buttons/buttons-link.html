Buttons link

UiBibz::Ui::Core::Buttons::ButtonLink

The button_choice component is an extension of button element.
Like this one, the button_choice component has options and html_options present as hash.
It inherits options of component element.
Make a button with a link (url).

Button link

Button link has several options:

  • status (:danger, :success, :info, :warning, :primary, :secondary)
  • size (:lg, :md, :sm, :xs)
  • state (:disabled, :active)
  • glyph (String || Hash)
  • outline (Boolean)
  • type (:block)
Primary Secondary Success Warning Danger Link
  • Ruby
  • Html
    button_link 'Primary', url: '#primary', status: :primary
    button_link 'Secondary', url: '#secondary', status: :secondary
    button_link 'Success', url: '#success', status: :success
    button_link 'Warning', url: '#warning', status: :warning
    button_link 'Danger', url: '#danger', status: :danger
    button_link 'Link', url: '#link', status: :link

    <a class="btn-warning btn" href="#warning">Warning</a>
    <a class="btn-danger btn" href="#danger">Danger</a>
    <a class="btn-link btn" href="#link">Link</a>
