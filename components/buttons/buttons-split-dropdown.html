Buttons split dropdown

UiBibz::Ui::Core::Buttons::ButtonSplitDropdown

The button_dropdown component is an extension of dropdown element.
Like this one, the button_dropdown component has options and html_options present as hash.
It inherits options of component element.
Use any button to trigger a dropdown menu by providing the proper menu markup.

Button split dropdown

Buttonsplit dropdown options

Dropdown split button has several options:

  • status (:danger, :success, :info, :warning, :primary, :secondary)
  • glyph (String || Hash)
  • size (:lg, :md, :sm, :xs)
  • state (:disabled, :active)
  • position (:right, :left)
  • type (:dropup, :dropdown)
