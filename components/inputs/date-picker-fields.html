Date Picker Fields

UiBibz::Ui::Core::Inputs::DatePickerField

This component is based on the library Jquery multiselect.

Date Picker Field

  • Ruby
  • Html
    date_picker_field 'search'
    
    <input type="text" name="search" id="search" data-date-locale="en" data-provide="datepicker" data-date-format="dd/mm/yyyy" data-date-today-btn="linked" class="date_picker form-control">
Documentation

Date Picker Field date options

  • status (:danger, :success, :info, :warning, :primary, :secondary)
  • state (:disabled, :active)
  • today_highlight (Boolean)
  • calendar_weeks (Boolean)
  • autoclose (Boolean)
  • dates_disabled (String || Array)

Date Picker Field with glyph

  • Ruby
  • Html
    date_picker_field 'date_at', append: glyph('calendar'), prepend: glyph('pencil')
    
    <input type="text" name="date_at" id="date_at" data-date-locale="en" data-provide="datepicker" data-date-format="dd/mm/yyyy" data-date-today-btn="linked" class="date_picker form-control">

Simple form

You can use the date_picker input gem with simple form. You must add this line to the "config/initializers/simple_form.rb"

  • Ruby
    ...
    config.custom_inputs_namespaces << "CustomInputs"
    ...

Then in your view , you can insert your input field date_pickerm.

  • Ruby
    simple_form(@search) do |f|
      ...
      f.input :date, as: :date_picker, append: 'Date'
      ...
    end
