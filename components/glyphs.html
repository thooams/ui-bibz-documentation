Glyphs

UiBibz::Ui::Core::Glyph

Includes over 634 glyphs in font format from the fontawesome.io set (version 4.6.2).

Glyph

  • Ruby
  • Html
    glyph 'diamond'
    
    <i class="glyph fa fa-diamond"></i>
Documentation

Size

L'argument size accepte pour argument un nombre ou :xs :md :lg

  • Ruby
  • Html
    glyph 'diamond', size: :xs
    glyph 'diamond', size: 2
    glyph 'diamond', size: :md
    glyph 'diamond', size: :lg
    
    <i class="glyph fa fa-diamond fa-1x"></i>
    <i class="glyph fa fa-diamond fa-2x"></i>
    <i class="glyph fa fa-diamond fa-3x"></i>
    <i class="glyph fa fa-diamond fa-5x"></i>
