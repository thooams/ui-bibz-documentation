Alerts, Notify

UiBibz::Ui::Core::Notify

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

Notify

Danger
Warning
Info
Success
  • Ruby
  • Html
    notify 'Danger', status: :danger
    notify 'Warning', status: :warning
    notify 'Info', status: :info
    notify 'Success', status: :success
    
    <div class="alert-danger alert alert-dismissible" role="alert">Danger</div>

    <div class="alert-info alert alert-dismissible" role="alert">Info</div>
    <div class="alert-success alert alert-dismissible" role="alert">Success</div>

Notify options

Success
Danger notification with link.
  • Ruby
  • Html
    notify 'Success', { status: :success, closable: true }, { class: 'my-alert' }
     
    notify({ status: :danger, closable: true }, { class: 'my-alert' }) do
      "Danger notification with <a class="alert-link" href="#">link</a>"
    end
    
    <div class="alert-success my-alert alert alert-dismissible" role="alert">Success<button type="button" class="close" data-dismiss="alert" aria-label="Close"><span aria-hidden="true">×</span><span class="sr-only">Close</span></button></div>

    <button type="button" class="close" data-dismiss="alert" aria-label="Close"><span aria-hidden="true">×</span><span class="sr-only">Close</span></button></div>
