Progresses

UiBibz::Ui::Core::Progress

Provide up-to-date feedback on the progress of a workflow or action with simple yet flexible progress bars.

Progress

0 25% 50
  • Ruby
  • Html
    progress 0
     
    progress "25%", { status: :primary, value: 25 }, { class: 'my-progress' }
     
    progress({ status: :primary, value: 50 }, { class: 'my-progress'}) do
      50%
    end
    
    <progress class="progress" max="100" value="0">0</progress>
    <progress class="my-progress progress-primary progress" max="100" value="25">25%</progress>
    <progress class="my-progress progress-primary progress" max="100" value="50">50
    </progress>
Documentation

Progress Options

Progress has several options:

  • status (:danger, :success, :info, :warning, :primary, :secondary)
  • value (Float) [required]
  • striped (Boolean)
  • animated (Boolean)

status

10 25 50 75 100

  • Html
    progress 10, status: :primary
    progress 25, status: :success
    progress 50, status: :warning
    progress 75, status: :danger
    progress 100, status: :info
